# Product Overview

Webベースの三目並べ（Tic-Tac-Toe）ゲーム。シンプルで直感的なUIで、2人のプレイヤーが交互に手を打つクラシックなボードゲームを提供します。

## Core Capabilities

- **ゲームプレイ**: 3×3のボードでXとOが交互に手を打つ
- **勝敗判定**: 横・縦・斜めの3つ並びを自動検出
- **ゲーム管理**: リセット機能とスタート画面への戻り機能
- **状態管理**: ゲーム開始/終了状態の管理

## Target Use Cases

- ブラウザで手軽に遊べるボードゲーム
- React学習プロジェクトとしての実装例
- シンプルなゲームロジックの実装パターン

## Value Proposition

- シンプルで理解しやすいコード構造
- モダンなReact + TypeScriptによる型安全な実装
- Viteによる高速な開発体験
