# 三目並べ（CLI版）

CLIベースの三目並べゲームです。

## セットアップ

```bash
npm install
```

## 実行方法

```bash
npm start
```

## 遊び方

1. ゲーム開始後、ボードが表示されます
2. 座標を「行 列」の形式で入力します（例: `0 1`）
3. プレイヤーXとOが交互に手を打ちます
4. 3つ並べば勝利です
5. 終了する場合は `q` を入力します

## 座標の例

```
  0   1   2
0 X | O | X
  ---------
1 O | X | O
  ---------
2   |   | X
```

上記の場合、`2 0` を入力すると左下のマスに手を打てます。
