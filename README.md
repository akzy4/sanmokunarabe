# 三目並べゲーム

Webベースの三目並べ（Tic-Tac-Toe）ゲームです。React + Viteで構築されています。

## セットアップ

```bash
npm install
```

## 実行方法

開発サーバーを起動:

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

## ビルド

本番用ビルド:

```bash
npm run build
```

ビルド結果のプレビュー:

```bash
npm run preview
```

## 遊び方

1. スタート画面で「ゲームを開始」ボタンをクリック
2. マスをクリックして手を打ちます
3. プレイヤーXとOが交互に手を打ちます
4. 3つ並べば勝利です（横・縦・斜め）
5. すべてのマスが埋まると引き分けです
6. 「リセット」ボタンでゲームをリセットできます

## 技術スタック

- React 18
- TypeScript
- Vite
- CSS3
